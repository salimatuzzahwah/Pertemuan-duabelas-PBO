# Pertemuan-duabelas-PBO
# Implementasi Tabbed Pane pada Project CRUD
Proyek ini merupakan hasil praktikum Pemrograman Berorientasi Objek (PBO) yang bertujuan untuk mengimplementasikan Tabbed Pane pada aplikasi CRUD (Create, Read, Update, Delete) menggunakan Java Swing dan JPA (Java Persistence API).

# Deskripsi
Tabbed Pane digunakan untuk menampilkan beberapa fitur aplikasi dalam satu jendela dengan sistem tab (seperti pada browser).
Setiap tab merepresentasikan fungsi CRUD yang berbeda menambah data, menampilkan data, memperbarui data, dan menghapus data sehingga tampilan aplikasi menjadi lebih terstruktur, interaktif, dan mudah digunakan.

# Fitur Utama

Create: Menambahkan data baru ke database.
Read: Menampilkan data dari tabel Kendaraan dan Pemilik.
Update: Memperbarui data yang sudah ada.
Delete: Menghapus data dari database.
Upload CSV: Mengimpor data dari file .csv.
Tabbed Pane UI: Navigasi mudah antar fitur CRUD dalam satu jendela.

# Konsep Utama: Persistence
Persistence adalah kemampuan aplikasi untuk menyimpan dan mempertahankan data secara permanen di dalam media penyimpanan seperti database.
Dalam proyek ini, data disimpan menggunakan Entity Class yang merepresentasikan tabel di database, dan JPA digunakan untuk mengelola interaksi data tanpa menulis query SQL secara langsung.
